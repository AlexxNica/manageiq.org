Wide Platform Support

ManageIQ supports many different platforms, including VMware, Hyper-V, OpenStack, oVirtm Amazon Web Services, Azure, Google Cloud Platform, Kubernetes, OpenShift and more.
{% include placeholder.html %}

Agentless

ManageIQ does not depend on agents and can discover pre-existing infrastructure. It is therefore extremely easy to install and operate.
{% include placeholder.html %}

SmartState Analysis

SmartState analysis allows ManageIQ to peek inside virtual machines, containers and hosts to discover their contents, without requiring an agent.
{% include placeholder.html %}

Virtual Appliance

ManageIQ is distributed as a single virtual appliance that is available for a wide variety of a platforms and is extremely easy to install.
{% include placeholder.html %}

Scalable Architecture

The ManageIQ virtual appliance can be deployed standalone, or as part of a federated cluster of many appliances.
